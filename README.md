## Authentification
### LOGIN 
route : /auth/login
method : POST

### Register
route : /auth/register
method : POST
HEADR : [content-Type: application/json]
## User
if you want to get all user:
route : /user
method : GET

and if you want find a user:
route : /user/:id
method : GET
